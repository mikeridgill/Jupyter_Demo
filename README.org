Jupyter Notebook and accompanying files to allow a demonstration, for new users of python. The Notebook has been created with the intention of being appropriate for scientists, who may already have experience of using [[https://www.mathworks.com/products/matlab.html][MATLAB]] or [[https://www.r-project.org][R]].

If presenting, the file =jupyter_demo.zip= can be shared with participants.  Alternativelty, this file is available to download at [[http://bit.ly/sos_python]].
